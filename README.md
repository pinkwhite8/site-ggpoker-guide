# site-ggpoker-guide

这是一个用于归档和发布多个独立 HTML 页面的仓库。仓库内容以静态 HTML 文件为主，不针对任何单个域名或具体网站。

## 项目简介

本仓库主要用于存放和发布一系列独立的 HTML 页面。这些页面可能涵盖不同主题或功能，但均以静态网页的形式提供。仓库本身不包含后台服务或动态内容，仅作为页面文件的存储与版本管理用途。

## 目录结构

```
site-ggpoker-guide/
├── pages/          # 存放主要 HTML 页面文件
├── assets/         # 页面引用的静态资源，如图片、样式表、脚本等
├── README.md       # 本文件
└── ...
```

- `pages/` 目录下为各个独立页面，每个页面通常为一个完整的 HTML 文件。
- `assets/` 目录存放页面所需的公共资源，按需组织子目录。

## 页面归档说明

本仓库中每个 HTML 页面均为独立归档，页面之间无强依赖关系。所有页面均可在浏览器中直接打开运行，无需额外服务器环境。页面内容可能包含说明性文字、工具界面或信息展示等，具体以页面内描述为准。

## 维护说明

- 本仓库由维护者不定期更新，新增或修改页面会通过提交记录体现。
- 欢迎提交 Issue 或 Pull Request 来报告问题或建议改进，但请注意保持内容与仓库主题一致。
- 仓库不提供任何形式的担保或支持，所有页面按原样提供。

## 许可

除非页面内另有声明，本仓库内容采用 MIT 许可协议发布。详情请参阅仓库根目录下的 `LICENSE` 文件（如存在）。
